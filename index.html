TaskManagerについて
TaskManagerは、あなたの予定を管理し、締め切りを通知するアプリです。本アプリは Google Calendar API を利用し、ユーザーの許可のもとで予定の登録・管理を行います。
Googleアカウントでログインし、Google Calendarと連携できます。登録した予定は、Google Calendar APIを使用して自動的にカレンダーへ追加されるだけでなく、編集や削除も可能です。
本アプリを開くと、締め切りまでの時間を視覚的に色や点滅で知らせてくれます。
通知を許可すると、締め切りが近づいた際に、ブラウザから「締め切りが近づいています」とリマインド通知が届きます。
